WB

The game rating is a rough read of this one game, not the player's Elo, so it shows as a provisional band rather than a single number. Checked against 64 real games, single-game estimates miss the true rating by several hundred points either way (a quiet game reads too strong, a sharp one too weak), and the band is only this game's own spread. Trends across many games mean something; one number does not.
